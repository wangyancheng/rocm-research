# MarkText 完整使用指南 (Markdown 教程)

> **提示**：在 MarkText 中，点击文字即可看到源码符号，按 `Ctrl + /` 可切换全篇源码模式。

---

## 1. 基础排版 (最常用)

### 标题

使用 `#` 号表示标题，支持 1-6 级：

- `# 一级标题` (快捷键: `Ctrl + 1`)
- `## 二级标题` (快捷键: `Ctrl + 2`)
- `### 三级标题` (快捷键: `Ctrl + 3`)

### 文本样式

- **加粗**：`**这里是加粗**` (快捷键: `Ctrl + B`)
- *斜体*：`*这里是斜体*` (快捷键: `Ctrl + I`)
- ~~删除线~~：`~~撤回的内容~~` (快捷键: `Ctrl + D`)
- `行内代码`：使用反引号 `` `code` `` (快捷键: `Ctrl + E`)

---

## 2. 列表与引用

### 无序列表

使用 `-`、`+` 或 `*` 加上空格：

- 项目一
- 项目二

### 有序列表

使用数字加点：

1. 第一步
2. 第二步

### 任务列表 (Todo List)

- [x] 已完成任务
- [ ] 待办事项

---

## 3. 插入多媒体

### 插入图片

语法：`![描述](路径)` (快捷键: `Ctrl + Shift + I`)
*技巧：直接从桌面拖拽图片进 MarkText，或者截图后 `Ctrl + V` 粘贴。*

### 插入链接

语法：`[百度](https://baidu.com)` (快捷键: `Ctrl + L`)

---

## 4. 代码与表格 (MarkText 强项)

### 代码块 (Bash/Python 等)

在空行输入三个反引号，后面跟上语言名称并回车。
例如输入：` ```bash ` 并回车。

### 可视化表格

输入 `/table` 或按 `Ctrl + T`，MarkText 会弹出行列选择器。
*技巧：表格上方有悬浮工具栏，可以快速增加行、调整对齐方式。*

---

## 5. 高级功能 (LaTeX & 图表)

### 数学公式

输入 `$$` 并回车：
$$
E = mc^2
$$

### 流程图 (Mermaid)

输入 ` ```mermaid ` 并回车，可以画出精美的流程图。

---

## 6. MarkText 独家提效快捷键

| 功能                    | 快捷键                |
|:--------------------- |:------------------ |
| **显示/隐藏侧边导航栏**        | `Ctrl + J`         |
| **快速跳转文章大纲**          | `Ctrl + Shift + L` |
| **切换源码模式**            | `Ctrl + /`         |
| **专注模式 (Focus Mode)** | `Shift + Ctrl + F` |
| **打字机模式**             | `Shift + Ctrl + T` |
| **万能搜索面板**            | `Ctrl + Shift + P` |

---

## 7. 避坑与建议设置

1. **自动保存**：前往 `File` -> `Preferences` -> `General` -> 勾选 `Auto Save`。
2. **图片路径**：在 `Preferences` -> `Image` 中设置 `Copy to designated folder`，防止文档移动后图片失效。
3. **关闭拼写红线**：在 `Preferences` -> `Spelling` 中关闭 `Enable spell check`。
